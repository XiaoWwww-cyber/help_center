# API 文档

## 概述

本文档介绍平台提供的 API 接口，帮助开发者进行系统集成。

## 认证方式

所有 API 请求需要在 Header 中携带认证信息：

```
Authorization: Bearer <your_token>
```

## 基础接口

### 获取用户信息

```
GET /api/user/info
```

### 数据查询

```
GET /api/data/query
```

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 500 | 服务器错误 |
