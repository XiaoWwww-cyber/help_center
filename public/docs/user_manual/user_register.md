# 用户注册指南

## 前言

欢迎使用本平台！本手册将帮助您快速完成用户注册和基础设置。

## 1. 用户注册

### 1.1 注册账号

1. 访问系统登录页面，点击【立即注册】按钮
2. 填写必填信息：用户名、密码、验证码
3. 系统提示"注册成功"后，即可使用账号密码登录

### 注意事项

- 用户名需全局唯一，不可重复注册
- 密码需符合安全要求（建议包含大小写字母、数字和特殊字符）

## 2. 忘记密码

1. 访问登录页面，点击【忘记密码】
2. 输入注册时使用的账号和手机号
3. 获取验证码并设置新密码
4. 点击【确认修改】完成密码重置

## 3. 修改密码

1. 登录系统后，点击右上角个人账号
2. 选择【修改密码】
3. 输入当前密码和新密码
4. 点击【确认】完成修改
